# 🔐 Password Strength Checker

A simple web-based tool to check the strength of a password.  
It validates whether the entered password contains:
- At least **8 characters**
- At least **one uppercase letter**
- At least **one lowercase letter**
- At least **one number**
- At least **one special character**

If the password meets all conditions, it is marked **Strong**. Otherwise, the user is prompted with guidelines to improve it.

---

## 🚀 Features
- Real-time password validation using **JavaScript & Regex**
- Clear feedback messages for **strong** and **weak** passwords
- Styled with **HTML + CSS** for a card-based UI
- Hover effects and shadows for a modern look
